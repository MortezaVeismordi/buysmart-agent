[![Status](https://img.shields.io/badge/status-active-success.svg)](#)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Problem
Procurement processes are often manual, slow, and inefficient. Businesses struggle to find the best deals, manage vendors, and analyze market trends effectively.

## Value Proposition
BuySmart Agent is an AI-powered autonomous procurement assistant that automates market research, vendor communication, and deal negotiation. It leverages LLMs and advanced crawling to provide actionable insights and execution capabilities.

## Quick Start
1. Clone the repo
2. Setup environment: `cp .env.example .env`
3. Run with Docker: `docker-compose up`
