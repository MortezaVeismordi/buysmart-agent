## Code of Conduct
We follow a standard open-source code of conduct based on respect, inclusivity, and constructive communication.  
Any behavior violating this will not be tolerated. Report issues to the repository maintainer.

## How to Contribute

### Reporting Bugs or Suggesting Features
- Use the GitHub Issues tracker.  
- For bugs: Include steps to reproduce, expected vs actual behavior, environment details (Python version, OS, etc.), and screenshots/logs if applicable.  
- For features: Describe the use case, proposed solution, and any alternatives considered.

### Contributing Code or Documentation
1. Fork the repository.
