# Repository Guide

This guide helps you navigate the BuySmart Agent repository in under 2 minutes.

## Directory Structure Overview

- `backend/`: Django core, domain logic, and agents.
- `frontend/`: Dashboard and UI components.
- `platform/`: Infrastructure (Docker, K8s, Observability).
- `docs/`: The "heart" of the narrative - start here for context.
- `configs/`: Runtime settings (environment, LLM, crawling).
- `data/`: Fixtures and demo data for development.
- `tools/`: Linting, formatting, and pre-commit scripts.
